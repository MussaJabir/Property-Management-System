What we completed in this period

1. Email delivery is live
The platform now sends email through the official no-reply@bjptechnologies.co.tz address. When BJP creates a new landlord client, the landlord automatically receives a welcome email with their sign-in link and a temporary password.
  • Delivered straight to the inbox (not spam), using the company's own mail server.
  • The welcome email was redesigned with proper PMS branding — a clean header, a clear sign-in details card, and a one-click "Sign in" button.
2. Data-privacy fix — each landlord sees only their own data
During testing we found that a landlord's staff panel could display another landlord's records. We traced the cause, fixed it, and confirmed the fix:
  • Every landlord workspace is now strictly separated — properties, units, renters, invoices and everything else are visible only to that landlord.
  • We added an automatic safety test that fails the build if this separation is ever weakened again, so it cannot silently return.
3. BJP control-panel dashboard
The super-admin panel now opens to a real overview instead of a blank page: total clients, active clients, clients on trial, total units across the whole platform, a breakdown of clients by plan, and a list of the most recently added clients.
4. Listings & panel polish
  • Each unit can now carry its own photo gallery, with a dedicated public page and a clean single-photo thumbnail in the staff list.
  • Units can be tagged with amenities (AC, Wi-Fi, water, parking, security, and more), which renters can filter by.
  • The landlord dashboard was reorganised to be glanceable, with quick status tabs and "return to the list after saving" for faster everyday work.

How we keep it safe & reliable

  • Strict separation between landlords' data
  • Automatic test guarding that separation
  • Email signed by the company domain
  • HTTPS with auto-renewing certificate
  • Nightly database backups
  • Sensitive tenant IDs stored encrypted
  • Every change reviewed before going live
  • Bilingual throughout (English & Kiswahili)

What's next

With the core platform live and hardened, the next focus is the Version 2 integrations already prepared in the system: mobile-money payments (M-Pesa, Tigo Pesa, Airtel Money), SMS and WhatsApp reminders, phone-number login for renters, and automatic monthly invoice generation. We'll schedule these together based on priority.
