Fixes & Improvements Report
Date: 8 June 2026
• Fixes & improvements

This report covers two changes delivered after the security programme: a data-privacy fix in the maintenance area, and an upgrade to how subscription plans are set up.

In short: when assigning a maintenance job, the staff list now shows only your own team — never another landlord's — and setting up a plan's features is now a simple tick-list instead of free-form typing.

What we changed

1. Maintenance — assign only to your own staff Privacy fix
What we found: when a landlord assigned a maintenance request to a staff member, the “Assign to” list mistakenly showed staff belonging to every landlord on the platform, not just their own. No-one outside PMS could see this, but one landlord could see the names of another landlord's team and (in theory) pick them.
  • The assignment list now shows only your own workspace's staff — everywhere a maintenance job can be assigned.
  • We added automatic tests that prove one landlord can never see or select another landlord's staff in that list.
  • As a small bonus, your team now appears in the list immediately, without having to type to search.
What it means: staff names stay private to each landlord, and maintenance jobs can only ever be assigned to your own people.
2. Subscription plans — pick features with checkboxes Improvement
On the BJP control panel, creating and editing subscription plans (Starter, Pro, Enterprise, and any new ones) is now faster and less error-prone. Instead of typing each plan's features as free-form text, you now tick the features a plan includes from a standard list:
  • Renter portal, website / CMS pages, maintenance tracking, reports & Excel export, email / SMS / WhatsApp notifications, priority support, and uptime SLA.
  • This works the same way as the landlord “Roles & Permissions” screen — consistent and quick.
  • You can still create brand-new plans and edit existing ones at any time.
What it means: plan setup is quicker and consistent. (For now these features are descriptive labels for the pricing page — they don't yet switch capabilities on or off; that can be added later if wanted.)

Quality & assurance

  • Maintenance staff list scoped to your workspace
  • Cross-landlord leak closed & tested
  • Plan features now a standard tick-list
  • Plan create & edit unchanged and working
  • Over 140 automatic tests run before release
  • Both changes peer-reviewed before going live
