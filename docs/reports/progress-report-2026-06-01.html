How we keep it safe and reliable

Each landlord is in their own private space
No landlord can ever see another landlord's tenants, invoices, payments, or expenses. Verified by automatic safety checks on every code change.
99 automatic safety tests
The system re-tests itself before every update. If anything important breaks, the update is blocked from reaching production.
Sensitive data is encrypted at rest
Tenant NIDA and TIN numbers are stored in encrypted form. Even with direct database access, the raw numbers cannot be read.
Money handled to the shilling
All money calculations use exact arithmetic. No rounding drift even on very large amounts, partial payments, or unusual billing cycles.
Invoice numbers never collide
Even if multiple staff issue invoices in the exact same instant, every invoice gets a unique sequential number per landlord per year.
Bilingual from day one
Every label, notification, and PDF supports English and Swahili. Users (landlord staff and tenants) can switch anywhere in one click.
Files stored on professional cloud storage
Property photos, lease PDFs, and receipts are stored on dedicated cloud storage (Backblaze B2) — never on the application server itself.
HTTPS, backups, and one-click rollback
Production runs behind HTTPS with auto-renewing certificates, nightly database backups, and a rollback script that reverts to the previous version in seconds if anything misbehaves.

What's next — Version 2

v1 is feature-complete and live. The next phase moves to the v2 integrations that were deliberately scaffolded but not switched on for v1:

  • Mobile money payments — M-Pesa, Tigo Pesa, Airtel Money via Selcom Pay (currently recorded manually)
  • SMS reminders — rent due, overdue, and payment confirmations via Beem Africa
  • WhatsApp receipts — receipts delivered directly to the tenant on WhatsApp
  • Login by phone OTP — tenants log in with phone number + one-time code
  • Automatic invoice generation — the system creates next month's invoices on its own
  • Live updates — dashboard refreshes as payments come in (Laravel Reverb / websockets)
  • Native mobile apps — Flutter apps for landlords and tenants, talking to the existing API
